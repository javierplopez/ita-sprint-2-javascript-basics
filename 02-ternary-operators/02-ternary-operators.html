2. Ternary Operators

⚠️ Important: Open the browser console to see the results

Exercise 1

Function:

                    
                        const puedeConducir = (edad) => edad >= 18 ? 'Puedes conducir' : 'No puedes conducir';

Example of usage:

                    
                        const result1 = puedeConducir(20);
                        console.log(result1); // Puedes conducir
                        
                        const result2 = puedeConducir(16);
                        console.log(result2); // No puedes conducir

Exercise 2

Function:

                    
                        const compararNumeros = (num1, num2) => num1 > num2 ? 'num1 es mayor' : 'num2 es mayor';

Example of usage:

                    
                        const result1 = compararNumeros(10, 5);
                        console.log(result1); // num1 es mayor
                        
                        const result2 = compararNumeros(3, 8);
                        console.log(result2); // num2 es mayor
